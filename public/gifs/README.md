# GIF Images for Quiz Questions

Place your GIF files in this folder following the naming convention:

- `question1.gif` — Question 1 meme
- `question2.gif` — Question 2 meme
- `question3.gif` — Question 3 meme
- `question4.gif` — Question 4 meme
- `question5.gif` — Question 5 meme
- `question6.gif` — Question 6 meme
- `question7.gif` — Question 7 meme
- `question8.gif` — Question 8 meme
- `question9.gif` — Question 9 meme
- `question10.gif` — Question 10 meme

These are referenced as `/gifs/question1.gif` etc. by Angular's static file serving.

The GIFs are displayed at the bottom of each question slide and can be toggled on/off using the 🎬 button visible on every page.
